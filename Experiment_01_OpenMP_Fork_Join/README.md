## 1. OpenMP Fork-Join Parallelism

### AIM:
To demonstrate OpenMP Fork-Join Parallelism.

### ALGORITHM:
Step 1: Start
Step 2: Include stdio.h and omp.h
Step 3: Print threads before parallel region
Step 4: Create parallel region
Step 5: Each thread prints its ID
Step 6: Print threads after parallel region
Step 7: Stop

### Source Code: fork_join.c
### OUTPUT: Thread IDs printed
### Result: Program executed successfully
